JSON Schema:
Array of override objects:

{ "collection": "Primitives", "variable": "Colors/Brand/500", "value": "#7a5af8" }

Fields:
• collection — name (substring match) of the variable collection
• variable — full variable path (e.g. Colors/Gray (light mode)/500)
• mode — (optional) mode name (e.g. Light mode, Dark mode). Defaults to first mode.
• value — hex color (#rrggbb or #rrggbbaa), number, or string

Value types:
• Colors: "#6938ef" or with alpha "#09090b1a"
• Numbers: 12, 16.5
• Strings/fonts: "Sora"
